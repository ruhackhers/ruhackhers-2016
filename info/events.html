Tech Talks

Microsoft Azure Tech Talk

Azure is Microsoft's cloud computing platform used for building, deploying, and managing applications. In this workshop, participants will learn how to integrate Azure into their hacks using web, mobile and APIs.

UPS Tech Talk

Learn about diversity within tech and how to prevent fraud.

Workshops

Introduction to Hacking

What exactly is a hackathon? Learn the basics of a hackathon, what it represents, and what to expect before we start actually hacking.

Introduction to APIs

Companies will be providing their APIs (Application Programming Interface) to hackers in order to create projects. Learn how to use these APIs and what kind of cool features you can add to your project.

Beginner Web Development

We're bringing back this pre-HackHERS workshop! Learn the basics of building a website and what tools you can use in order to make beautiful and functional websites.

Life After College

Hear what current graduate students, professors, and professionals in tech have to say about their experiences.

Android Workshop

Learn how to build an android mobile application! Basic Java knowledge is required. Please download Android Studio beforehand, if you plan to attend.

Git and Hosting

Git is a version control system which is a useful resource for coders. Come to learn basic Git commands and how to host your website.

Introduction to Pebble Development

Interested in wearable technology? Want to make apps for a smart watch? Learn how to use Pebble.js and build basic Pebble applications, even if you don't own a Pebble!

Fun

Cup Stacking

A hackathon classic! Competitors will stack and unstack cups for 1 minute and 30 seconds. The person who stacks/unstacks the most times will be the winner. Winners for each round will compete against each other in final round to determine final winner of prize.

Jigsaw Puzzle

Jigsaw puzzle will be left out throughout the Hackhers event for people to put pieces down (Hopefully it will be completed by the end of the event! :D)

Cupcake Decorating

Nothing says treat yo'self like decorating delicious cupcakes and eating them right after.

Marbelizing Magnets

Ever find a picture that you really like and wish that you could make it into a magnet? Don't worry, we got you. Take a break from hacking and join us for this arts and crafts activity!

Mod Podge Craft

Another arts and crafts activity! Come make plant holders or pencil holders!

Improv

Take a break from hacking and play some theatre and Who's-Line-is-It-Anyway-esque games. Perfect if you need to move and get the blood flowing after hours of staring at a computer.
